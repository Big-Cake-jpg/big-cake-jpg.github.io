## Usage | 用法

```bash
git clone https://github.com/Big-Cake-jpg/big-cake-jpg.github.io.git
yarn # npm install
valaxy new 
yarn build
```

### Publish | 日常发布

推送至 GitHub，使用 Actions 生成静态页面并推送 main 分支
